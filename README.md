# pokeGAN

## Dependencies (pip install) 
```
cv2
tensorflow( >=1.0)
scipy
numpy
```
## Usage
```
cd pokeGAN
python resize.py
python RGBA2RGB.py
python pokeGAN.py
```

And open the notebook by typing in `jupyter notebook` in the root directory. Install it [here](http://jupyter.readthedocs.io/en/latest/install.html) if you haven't
